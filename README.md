# Skeleventy

A simple boilerplate built with Eleventy, TailwindCSS and Parcel.

## Features

- Build sites faster with the power of [Eleventy](https://www.11ty.dev/docs/), [TailwindCSS](https://tailwindcss.com) and [Parcel](https://parceljs.org/)

## Requirements

Node `>=` v8.9.0

## Installation

```
npm install
```
